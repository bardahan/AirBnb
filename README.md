## AirBnb

- Web App dev finel project

### Install

Make sure you have mongoDB installed!

On main project folder run:

<code>
npm instsall
</code>

### Run

On main project folder run:

<code>
npm start
</code>

### Local Website Address

<code>
http://localhost:3000
</code>
